# temps-sommeil
HTML 5 – CSS 3 – JQuery – SQL – PHP – Bootstrap 

http://temps-sommeil-xyz.mon.world/


Application web pour enregistrer son temps de sommeil

La sécurité est faible et il n'y a pas d'authentification puisque son usage est prévu pour être totalement privé et accessible uniquement en réseau local.

La version en ligne du site est une démo avec de fausses données. 
Vous êtes donc libre d'ajouter ou de modifier des nuits de sommeils enregistrées.
